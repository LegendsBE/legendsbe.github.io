+ Stating the same message in chat 3 or more times in a row with no change / Repeating 3 or more of the same message in one chat box.
 ~ “Buy my pickaxe” ~  “Buy my pickaxe!” ~  “Buy my pickaxe!” 
+ Spamming a staff member in a private message with 10 or more messages in a 1-minute time frame.
 
+ Spamming commands that are viewable via other players.
  ~ /tell Spam ~ /Island Cooperate Spam  ~ /Pay Spam 
+ Sending 3 or more of the same text in a single chat message.
  ~ “Hi hi hi.” ~ “Hello hello hello!”
